# FSTR API

API для Федерация Спортивного Туризма России (ФСТР)

## Описание

Этот API позволяет:
- Добавлять данные о новых перевалах
- Получать информацию о перевалах
- Редактировать данные (если статус "new")
- Просматривать все перевалы пользователя
